# Iconography
Additional documentation available at [orion.united.com](https://orion.united.com/)
#
## Orion icon fonts usage
`<i class="orion-icon orion-icon-[name]"></i>`  
 
**Example:**  
`<i class="orion-icon orion-icon-alert"></i>`  
#   
## Icon sizing
Size can be set by the font-size style on the parent container or on the icon element explicitly  
  
**Example:**  
`<i style="font-size:30px" class="orion-icon orion-icon-alert"></i>`  
#
## Icon color
Color can be set by the color style on the parent container or on the icon element explicitly  

**Example:**  
`<i style="color:red" class="orion-icon orion-icon-alert"></i>`  
#
## Icon rotation
Can be done by the app developer using custom styles or leveraging the following orion utility classes:  
`rotate-45`, `rotate-90`, `rotate-180`  
  
**Example:**  
`<i class="orion-icon orion-icon-teen rotate-90"></i>`  
#
## Icon flipping
Can be done by the app developer using custom styles or leveraging the following orion utility classes:  
`flip-x`, `flip-y`  
  
**Example:**  
`<i class="orion-icon orion-icon-teen flip-y"></i>`
